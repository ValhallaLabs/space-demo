# space-cource
This is simple space arcade build with Three.js

### Technologies:
  - Node.js; 
  - Express.js;
  - Jade;
  - ES6;
  - Three.js;
  
# How to run server

Install dependencies with `npm install` and then run server `npm start`, open http://localhost:3000. Enjoy!

````
node -v
  v6.10.3
npm -v 
  3.10.10
````
